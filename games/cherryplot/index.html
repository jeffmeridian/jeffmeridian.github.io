Industrial Features

"The machine comes fully equipped for industrial-scale word production."

Binder & Organization

  • Hierarchical Structure: Organize content into infinite nested Folders and Text documents.
  • Drag & Drop: Easily reorder documents and restructure your project via the Binder.
  • Search & Replace: Global search and batch replace functionality across the entire project.
  • Custom Icons: Custom icons for folders and documents.

Writing Experience

  • Distraction-Free Editor: Clean interface with focus mode.
  • Teleprompter Mode: Adjustable scrolling text for reading or presenting.
  • Inspector: Manage metadata like Status, Synopsis, and Research Notes. Upload images to the inspector (used with save epub).
  • History Manager: Robust Undo/Redo stack to protect your work.
  • Auto-Save: Changes are automatically persisted to the local SQLite database.
  • Dark Mode: Toggleable dark/light theme for comfortable writing.

Import & Export

  • Scrivener Import: Import entire .scriv projects. Automatically parses Binder structure, synopses, and notes.
  • EPUB Support: Generate production-ready EPUBs directly or view external files with the built-in reader.
  • Project Backup: Export/Restore the entire database as a JSON backup.
  • Markdown Export: Export the entire project as a markdown file.

Integrated Agent

  • Built-in Chat: Interface for AI assistance while writing.
  • Voice Interaction: Chat agent speaks with adjustable speed and a large selection of Kokoro AI voices.
  • Interactive Mode: Toggle interactive mode to enable continuous voice conversation.
  • Chat Management: Save your chat sessions for later reference.
  • Chat Memory: The chat has a short memory for context.
