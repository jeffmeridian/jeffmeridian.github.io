User Guide: Creating Postings

This guide explains how to create and customize postings for the Writer Studio Eleventy site.

↑ Back to Top

1. Creating a New Post File #

Posts are stored in a date-based directory structure for organizational clarity:
eleventy/site/YYYY/MM/DD/your-post-title.md

Example: eleventy/site/2026/03/15/my-new-story.md

↑ Back to Top

2. Front Matter Configuration #

Every post must start with a YAML front matter block.

Basic Example

---
title: "Your Post Title"
date: '2026-03-15T00:00:00Z'
authors:
  - name: your name
metadata:
  categories:
    - short stories  # or: video-blog, app, etc.
  media:
    featuredImage: assets/your-image.webp
    imageAlt: 'Description for accessibility'
tags:
  - english         # Use: english, deutsch, or espanol
  - short stories
---

Language Tags

The site uses tags to decide which version of the homepage to show the post on:

  • English: english or en
  • German: deutsch or german
  • Spanish: espanol, español, or es

Images with Captions

Wraps an image in a proper <figure> tag with a caption.

<figure class="post-image-container">
      <img src="/assets/photo.webp" alt="A beautiful sunset over the mountains">
      <figcaption>A beautiful sunset over the mountains</figcaption>
    </figure>

4. Layout Features #

Table of Contents

To automatically generate a list of links to all your ## Headings, add this marker:

[[toc]]

Back to Top Links

The system automatically adds a "↑ Back to Top" link before every ## Heading.
Requirement: Add this empty marker near the top of your post to give the link a destination:

<div id="toc-container"></div>

Standard Markdown Highlights

  • Auto-linking: Just paste a URL and it becomes clickable.
  • Header Anchors: Every heading automatically gets an ID for direct linking.
  • Standard Formatting: **Bold**, *Italics*, [Links](url), > Quotes.

5. Assets & Images #

  • Store images in an assets/ subfolder next to your .md file.
  • It is recommended to use WebP format for better performance.
